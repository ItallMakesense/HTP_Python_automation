"""
Send me the sequence of shell commands below. Mind that all the commands should be called from user’s “home” directory:

o   Create a directory named after your first and second names separated by space, aka “Vassili Ivanov”;

o   Create an empty file named “-current-date-file”

o   Add current date to the file content

o   Create a hard link to that file in the directory with your name

o   Remove existing “-current-date-file” from home directory

 

*As a task to gain several extra points:

Study <mount>, <fdisk> and <mkfs> commands.

Create one additional HDD in VirtualBox, partition it with fdisk, create one ext4-formatted partition and mount it permanently to /mnt dir.

Solution:
0) cd /home
1) mkdir Evgeny\ Kirillov
2) cat > -current-date-file
3) date > -current-date-file
4) ln ./-current-date-file /home/Evgeny\ Kirillov
5) rm ./-current-date-file
Extra:
0) Settings > Storage > Controller: SATA > Add hard disk > Create new disk > VDI > Fixed Size > Create
1) fdisk -l    # checking
2) fdisk /dev/sdb
3) : n
4) : p
5) : enter    # first cylinder (default 1)
6) : enter    # last cylinder (default 1044)
7) : w
8) fdisk -l    # checking
9) mkfs -t ext4 /dev/sdb1
10) mount /dev/sdb1 /mnt
